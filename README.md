# C-Compiler

仓库结构
--------------
```
C-Compiler/
├─ README.md                
├─ draw/                    # DFA 与流程图（.drawio）
│  ├─ DFA-保留字.drawio
│  ├─ DFA-界符.drawio
│  └─ flow-保留字和标识符识别.drawio
└─ src/
	├─ c-code.txt            # 示例 C 代码输入
	└─ Lexer.py              # 词法分析器实现
```

## 环境

根目录有 .venv 文件夹 

```shell
source .venv/bin/activate
```
